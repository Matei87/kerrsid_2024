#kerrsid_2024

## How to Run

1. Clone the repository:

   ```sh
   git clone https://github.com/Matei87/kerrsid_2024.git
   cd kerrsid_2024
   ```

2. Install the dependencies:

   ```sh
   npm i
   ```

3. Run the application:

   ```sh
   npm run dev
   ```

4. Open the app in browser

   ```sh
   http://localhost:5173
   ```
